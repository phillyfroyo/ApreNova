## **üß† GPT-Powered Translation**

    Optional example sentences for deeper learning
    
- **API Endpoints:**

- **Modes:**

- **Bilingual Support:**
    
    Works on both `/es/stories` (English ‚Üí Spanish) and `/en/stories` (Spanish ‚Üí English)
    
    Auto-detects direction and selects appropriate GPT prompt
    
    Prompts dynamically adjust based on user‚Äôs level (L1‚ÄìL5)
    
- **Example Sentences:**
    
    Optional GPT-generated examples for each translation
    
    Matched sentence pairs: Spanish + English
    
    Pulled on demand and cached
    
- **Caching:**
    
    In-memory map (ready for Redis or Vercel KV)
    
- **Output Handling:**
    
    GPT responses are structured as `{ primary, otherCommonTranslations }` and parsed with fallback/error handling. Phrase and word prompts enforce contextual awareness for primary output, with optional general-use alternatives.

### üåç **Internationalization & Language Strategy**

- **Language Routing:** Dynamic `[lng]` param (`/en`, `/es`) powers all core routes
- **UI Translations:** Centralized key-value dictionaries (`content/ui/en.ts`, `es.ts`)
- **Translation Helper:** `t(lang, section, key)` fetches UI strings from nested dictionaries (`content/ui/en.ts`, `es.ts`)
- **Language Memory:**
    - Stored in `localStorage` as `preferredLang`
    - Auto-redirect logic on `/` if known
- **Language Selection Page:** Root `/` route prompts for native language
- **User-Based Language:** On login, users redirected via `session.user.nativeLanguage`
